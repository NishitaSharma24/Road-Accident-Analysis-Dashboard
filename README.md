# 🚗 Road Accident Analysis Dashboard

An interactive Power BI dashboard developed to analyze road accident data and uncover patterns related to accident severity, casualties, police attendance, environmental conditions, vehicles, road types, and accident trends. The dashboard enables users to explore insights through dynamic visualizations and interactive filters.

---

# 📌 Dashboard Overview

This project provides a comprehensive analysis of road accidents using Power BI. It helps identify accident trends, casualty distribution, environmental impacts, police response, and vehicle involvement to support data-driven decision-making.

---

# ✨ Dashboard Features

- Interactive Power BI Dashboard
- Dynamic Filters (Weather, Area Type, Severity & Month)
- KPI Cards
- Interactive Maps
- Trend Analysis
- Drill-down Visualizations
- Business Insights

# 💡 Key Insights

- Slight severity accidents accounted for the majority of total accidents.
- Urban areas recorded significantly higher accident counts than rural areas.
- Most accidents occurred during daylight conditions.
- Police attended over 80% of reported accidents.
- Accident frequency varied across different speed limits and road types.
- Casualty trends changed across months, weekdays, and weather conditions.
- Vehicle involvement was highest in slight severity accidents.
